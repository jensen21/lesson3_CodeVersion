Highlight #1 Version Control System

It is a main repository for all project files. Developers are able to pull a file and then make changes. After the changes are complete, the developer can commit the files back to the repository.The version control system notes when files were changed, who made those changes, merges old and new files, and does so much more.

Highlight #2 Git

It is an open source distributed version control system that is made to handle small and large projects.Git is efficient and very fast. Some benefits of using git is that it is local, each user has a local copy of the stored data in the repository, and it prevents unintentional overwrite of any files.

Highlight #3 Git Commands

In this class, we use git commands in codio to communicate between codio and git. We have three main commands for this assignment. The three commands are git add ., git commit, and git push.
